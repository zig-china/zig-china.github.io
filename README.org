本网站使用 [[https://gohugo.io/][hugo]]（extended 版本）与 [[https://www.docsy.dev/][docsy]] 主题进行构建。
预览之前请确保已安装必要的 =postcss= 依赖
#+begin_src bash
# install depends
npm i -D postcss postcss-cli autoprefixer
#+end_src

#+begin_src bash
# For macOS
brew install hugo
# For Debian
sudo apt install hugo
# For Arch
sudo pacman -S hugo

# Snap
sudo snap install hugo
#+end_src

#+begin_src bash
# run server
hugo server
#+end_src
